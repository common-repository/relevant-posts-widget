== Description ==

The WordPress Relavant Posts Widget plugin provides a widget to show relevant posts to the current displayed post.  You can set the options in the widget form and easily add the widget to your sidebar.

Now , The plugin supports 2 ways to refer relevant posts, in same category or with same tags.  Maybe in the future, some more algorithms to find relevant posts will be added based upon your suggestions. So please leave comments to me if you have any good idea.

Features:

* Show posts within same category.
* Show posts sharing the same tags.
* Options to show 2 kinds of relevant post or just one kind.
* Options to set how many posts to show seperately.
* Automatically remove duplicate posts.
* Show number of comments.

== Installation ==


1. Upload plugin directory to the '/wp-content/plugins/' directory. Or you can just download and install it through the 'Plugins' menu in your WordPress administration panel. 
2. Activate the plugin.
3. In widget section,  pull the 'Relavant Posts' widget to sidebar.
4. Set options in widget configuration form.
5. Save and enjoy it!
